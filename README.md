# pdf2word_translater
pdf 에 포함되어 있는 수식까지 완벽하게 word 로 변환하며, 이후 llm을 통한 번역을 수행합니다.

yolo를 이용한 pdf 레이아웃 탐지, 수식을 latex로 전환 후 양식에 맞춰 붙여 넣습니다.
